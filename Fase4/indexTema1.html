Fundamentos de HTML

Para empezar, debemos decir que es HTML, sus siglas que significan (HyperText Markup Language), en español es lenguaje de marcas de hipertexto y es el componente mas básico de los sitios web. Este forma lo que va a ser la estructura del contenido web del sitio, el HTML usa marcas o también conocidas como etiquetas, para textos, imágenes y demás tipo de contenido para mostrarlo en el sitio y posicinarlo. Algunas de estas marcas son head, title, body, header, footer, canvas, article, p, div, section, form, img, video, iframe, nav, entre otras las cuales se tienen que poner entre dos signos de menor que y maor que, de esta forma "<" ">", y en algunos se deben cerrar con el mismo nombre de etiqueta y los signos de menor y mayor pero con un back-slash "/" antes del nombre de la etiqueta y despues del primer signo.

Los elementos de HTML son los bloques de construcción en una pagina web, estas etiquetas o elementos le dicen a los navegadores donde inicia y termina un elemento.

Las partes de un elemento son:

  • Etiqueta de apertura: Son las que se utilizan para indicar donde un elemento inicia y funciona.
  • Contenido: Es lo que ira dentro de las etiquetas.
  • Etiqueta del cierre: Similar a la de entrada pero con un "/" para terminar el elemento.

Otra parte fundamental en HTML son los atributos, estas constan con un nombre y un valor, el nombre define la información de mas que el usuario quiera agregar, y el valor del atributo da mas especificaciones. Estos se ponen dentro de la etiqueta.

Esto es un ejemplo de lo anterior aplicando el atributo style, con el nombre "color" y el valor "yellowgreen" y fuente verdana con "font-family: verdana".

Otro atributo son las clases, este atributo tambien es importante ya que añade mas información a los estilos o si usamos librerias como bootstrap para hacer las web responsivas o adapativas para telefonos moviles.

Algunas etiquetas no son necesarias cerrarlas como las de imagen "img"
